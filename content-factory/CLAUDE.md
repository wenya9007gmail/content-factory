## To get started
- Clone your repo locally and start building using your current editor and terminal
- `git status` to see what files you've changed and `git add <filename>` to stage a file
- Run `npm test` or `npm start` to test your code and `npm run lint` to lint
- Finally, run `git commit` to save your changes in your local repo.
